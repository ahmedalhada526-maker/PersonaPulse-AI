# نبض الشخصية (PersonaPulse AI) - خبير علم النفس السلوكي والتواصل الاستراتيجي

نبض الشخصية هو منصة ذكاء اصطناعي متطورة مصممة لتحليل الأنماط اللغوية، وفك شفرات النصوص الخفية، وتصميم استجابات استراتيجية باستخدام علم النفس السلوكي.

## المميزات

- **التحليل الجنائي اللغوي:** تحديد سمات الشخصية (MBTI، الخمسة الكبار، إنياغرام).
- **فك تشفير النصوص الخفية:** كشف النوايا المخفية والمحفزات العاطفية.
- **هندسة الاستجابة الاستراتيجية:** توليد ردود مخصصة لأهداف محددة.
- **نموذج Freemium:** 10 عمليات مسح مجانية يومياً مع خيار الترقية إلى Pro.
- **دعم متعدد اللغات:** دعم كامل للغتين الإنجليزية والعربية (RTL).
- **واجهة مستخدم فاخرة:** وضع ليلي سينمائي مع تأثيرات Glassmorphism وتخطيط Bento Grid.

## البدء بالعمل (Setup Guide)

### المتطلبات الأساسية
- **Node.js**: يوصى بالإصدار 18 أو أعلى.
- **مدير حزم**: npm أو Yarn.

### التثبيت (Installation)
1. استنسخ المستودع (إن وجد).
2. قم بتثبيت التبعيات باستخدام الأمر التالي:
   ```bash
   npm install
   ```

### الإعدادات (Configuration)
1. أنشئ ملفًا باسم `.env.local` في الدليل الجذر للمشروع.
2. أضف مفتاح واجهة برمجة تطبيقات Gemini الخاص بك داخل الملف:
   ```env
   GEMINI_API_KEY=your_actual_api_key_here
   ```

### تشغيل التطبيق (Running the App)
1. ابدأ تشغيل خادم التطوير:
   ```bash
   npm run dev
   ```
2. سيكون التطبيق متاحًا على الرابط: `http://localhost:3000` (أو المنفذ المحدد في وحدة التحكم الخاصة بك).

## النشر على GitHub (GitHub Deployment)

لاستضافة هذا المشروع على GitHub Pages مجاناً:

1. قم بإنشاء مستودع (Repository) جديد على GitHub وارفَع الكود إليه.
2. من إعدادات المستودع، اذهب إلى **Settings > Secrets and variables > Actions**.
3. أضف "New repository secret" باسم `GEMINI_API_KEY` وضَع مفتاحك هنا.
4. اذهب إلى **Settings > Pages** وتأكد من ضبط "Build and deployment" لاستخدام **GitHub Actions**.
5. سيتم بناء الموقع ونشره تلقائياً عند كل عملية رفع (Push) لفرع `main`.
